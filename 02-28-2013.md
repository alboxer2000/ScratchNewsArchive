## Choose a color

![](https://64.media.tumblr.com/615e6f58c350deb34fdfc2615d25ec51/85718c430b0bfce5-05/s100x200/0f469d32e320e8af8728429a82f0e5e0f1a45aa7.png)

As part of our [paint editor redesign](http://beta.scratch.mit.edu/forums/topic/2649/), we’ve updated how [to change colors](http://beta.scratch.mit.edu/projects/10063025/).
We are working to add the ability to switch between bitmap and vector tools. As a step towards this, in the Paint Editor, you now just choose one color at a time, which you can then apply to change any fill or border. See [this project](http://beta.scratch.mit.edu/projects/10063025/) to learn more.

Note that this fix only applies to the current vector editor. We’re still working on a bitmap editor option, which we hope to have ready for testing on the beta in a few weeks time. (And by the way - gradients are planned for the future).
